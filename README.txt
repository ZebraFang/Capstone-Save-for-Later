Project name: Information/hobby website with a save for later function.

2)
Description: This project gives a short insight into web development and it has a save for later function using local storage.

3)
To set up the files just clone repo and make sure that the html, css and js are in the same directory. Images should be in their own directory. Once set up simply open index.html with a web browser

4) To use the save for later function simply click on an article you want to read later and it will populate on the saved page. To access the saved page click the link on the navbar. The navbar will navigate you to other sections of the website for your perusal.
